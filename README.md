# PubQuest Web

Web client for PubQuest — a social pub-crawl app with RPG elements. Users can check in to venues, earn gold, level up, form parties, manage friends, and explore a live map with venue and friend markers.

Built with Next.js 16, React 19, Leaflet, TanStack Query, React Hook Form, Zod, and Tailwind CSS.

## Prerequisites

- Node.js 20+
- [pubquest-api](../pubquest-api) running locally on port 3000

## Getting Started

```bash
npm install
npm run dev
```

The app runs on [http://localhost:3001](http://localhost:3001) (Next.js will use the next available port since the API occupies 3000).
